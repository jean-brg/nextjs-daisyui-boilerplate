# NextJS DiasyUI Boilerplate
A simple boilerplate based off `create-next-app` with Tailwind and DaisyUI's component library.
Nothing fancy, just a little code to make set up faster.

## Instalation
1. Clone the repo
```bash
git clone <url>
```

2. Install packages
```bash
npm install
```
